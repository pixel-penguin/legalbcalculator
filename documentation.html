Troubleshooting

Common Issues

Widget not displaying

  • Ensure the container ID is exactly calculator-widget-transfer-cost-calculator
  • Check that the script is loaded correctly (no 404 errors in browser console)
  • Make sure there are no JavaScript errors preventing execution

Calculation not working

  • Check browser console for network errors
  • Ensure all form fields are filled in
  • Verify that the purchase price is a valid number

Styling issues

  • Check for CSS conflicts with existing stylesheets
  • Use more specific CSS selectors with !important if needed
  • Test in different browsers to ensure compatibility
