Legal Calculator Widget

Property Transfer Cost Calculator for Namibia - Integration Guide & Documentation

Overview

The Legal Calculator Widget is a JavaScript component that allows you to integrate property transfer cost calculations directly into your website. It provides an easy-to-use interface for calculating transfer costs based on Namibian property law.

Quick Start

Get started with the Legal Calculator Widget in under 5 minutes:

1. Include the Widget Script

<script src="https://your-bucket-url.s3.amazonaws.com/legal-calculator-widget.js"></script>

2. Create a Container

<div id="legal-calculator"></div>

3. Initialize the Widget

<script> document.addEventListener('DOMContentLoaded', function() { new LegalCalculatorWidget('legal-calculator', { apiUrl: 'https://your-api-gateway-url.execute-api.region.amazonaws.com/sandbox/calculate', theme: 'light', currency: 'N$' }); }); </script>
That's it! Your calculator is now ready to use.

API Reference

POST /calculate

Request Parameters

Parameter Type Required Description
amount Number Required Property value in Namibian Dollars
sub_type String Required Property type: "S" (Sectional Title) or "F" (Freehold)
dutytype String Required Duty type: "N" (Normal/Residential), "A" (Agricultural), "C" (Commercial)
date String Required Transfer date: "before" (before Oct 1, 2024) or "after" (from Oct 1, 2024)

Response Format

{ "transferFees": 12000.00, "vatOnFees": 1800.00, "transferDuty": 5000.00, "stampDuty": 1200.00, "deedsOfficeFee": 400.00, "sundriesPostagesVAT": 1265.00, "total": 21665.00 }

Configuration Options

Option Type Default Description
apiUrl String sandbox URL API endpoint URL
theme String 'light' 'light' or 'dark'
currency String 'N$' Currency symbol to display
loadingText String 'Calculating...' Loading message text
errorText String 'Error calculating costs. Please try again.' Default error message
containerClass String 'legal-calc-widget' CSS class for the main container

Styling & Customization

Built-in Themes

The widget comes with two built-in themes:

Light Theme (Default)

new LegalCalculatorWidget('calculator', { theme: 'light' });

Dark Theme

new LegalCalculatorWidget('calculator', { theme: 'dark' });

Custom Styling

You can override the default styles by targeting the widget's CSS classes:

/* Custom styling example */ .legal-calc-widget { border: 2px solid #your-brand-color; border-radius: 15px; font-family: 'Your Custom Font', sans-serif; } .legal-calc-widget .calc-button { background: #your-brand-color; border-radius: 25px; } .legal-calc-widget .calc-header h3 { color: #your-brand-color; }

Troubleshooting

Common Issues

Widget not displaying

  • Check that the script is loaded before initializing the widget
  • Ensure the container element exists in the DOM
  • Check browser console for JavaScript errors

API errors

  • Verify the API URL is correct
  • Check CORS settings on your API Gateway
  • Ensure all required parameters are being sent

Styling issues

  • Check for CSS conflicts with existing stylesheets
  • Use more specific CSS selectors to override styles
  • Ensure the theme parameter is set correctly

Debug Mode

Enable debug mode to see detailed error messages:

new LegalCalculatorWidget('calculator', { apiUrl: 'your-api-url', debug: true });
